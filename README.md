# BuildFlow AI

BuildFlow AI is an AI-based project planning assistant that helps users build complete projects using structured AI guidance.

## Features
- Guided AI planning chat across structured stages
- Blueprint generation from conversation context
- Document generation (BRD, SRS, Roadmap, TODO)
- Prompt generation for development workflows
- Firebase authentication and per-user project data

## Tech Stack
- Frontend: React, Vite, Tailwind CSS
- Backend: Node.js, Express
- AI: OpenRouter integration
- Auth/Data: Firebase Auth and Firestore

## How to Run
### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
node server.js
```
